# Changelog

Todas as mudanças notáveis neste projeto serão documentadas neste arquivo.

## [1.0.0] - 2025-04-11

### Adicionado
- Interface CLI interativa com menu numerado
- Funcionalidade de adicionar gastos com descrição, valor, categoria e data
- Listagem de gastos com filtro opcional por categoria
- Remoção de gasto por ID
- Resumo de gastos agrupados por categoria
- Cálculo do total gasto
- Persistência de dados em arquivo JSON local
- Testes automatizados com pytest (13 testes)
- Análise estática com ruff
- Pipeline de CI com GitHub Actions
